vmware安装vmware-tools

vmware网络设置

vi /etc/sysconfig/network-scripts/syscfg-enoxxxxx
boot 改为 true
service network restart

安装库

yum install gcc perl kernel-devel
yum search ifconfig
yum install net-tools.x86_64

更新

yum -y update
reboot

安装tools

载入安装目录下linux.iso
mount /dev/cdrom /media
cd /media
cp VMwareTools-xxxxx.tar.gz /home
cd /home
tar -zxcf VMwareTools-xxxxx.tar.gz
cd /vmware-tools-distrib/
./vmware-install.pl
一路回车
reboot
